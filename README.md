# AL RAMZ MEP DESIGN — Official Website

Official corporate website for **AL RAMZ MEP DESIGN**, a premier Mechanical, Electrical, and Plumbing (MEP) engineering consultancy with over 24 years of experience delivering high-performance engineering design, authority-approved shop drawings, site surveys, BOQ, and calculations in the UAE and India.

---

## 🌟 Key Features

* **Modern Luxury Design**: Built with a sleek dark obsidian theme, editorial typography, custom cursor system, and fluid micro-animations.
* **Interactive Lightbox Modal**: Click any project card to open a full-resolution high-contrast photo preview with captions.
* **WhatsApp Form Integration**: Contact form automatically formats client inquiry details (Name, Phone, Email, Service, Project Description) and redirects directly to WhatsApp (`+971 50 362 0281`).
* **Floating WhatsApp Action**: Quick-access floating WhatsApp chat button available across all pages.
* **Projects Portfolio**: Categorized project filter grid featuring landmark projects (Cartier Boutique, Prada Dubai Mall, Villas in the Hills, AYA Universe, Jones the Grocer, University of Balamand, etc.).
* **Fully Responsive**: Optimized for seamless performance on Desktop, Tablet, and Mobile devices.
* **Zero Dependency Core**: Pure Vanilla HTML5, CSS3, and JavaScript (GSAP & ScrollTrigger for animations).
